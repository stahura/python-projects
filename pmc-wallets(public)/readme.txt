The plan was to make a .command file to double click and run the script, but didn't have time.

The gold feed comes from https://metals-api.com/. They have a few options, I chose the free one and the API KEY can only be used 50 times a month.

The API Calls and key for the wallets come from etherscan.io



Before running the py file, make sure to activate the virtual environment:
	
	> pmcdata-env/bin/activate
	> python3 pmcdata.py



We could write the information to another csv file or whatever we want.
